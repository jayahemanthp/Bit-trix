### Butterfly Unit (Radix-2 DIT)
┌──────────────────────────────────────────────────────────────┐
│ Butterfly Unit │
├──────────────────────────────────────────────────────────────┤
│ │
│ A_real ────┐ │
│ A_imag ────┼────────────────────────┐ │
│ │ │ │
│ B_real ────┼──┐ │ │
│ B_imag ────┼──┼──┐ │ │
│ │ │ │ │ │
│ W_real ────┼──┼──┼──┐ │ │
│ W_imag ────┼──┼──┼──┼──┐ │ │
│ │ │ │ │ │ │ │
│ inv ───────┼──┼──┼──┼──┼──┐ │ │
│ │ │ │ │ │ │ │ │
│ ▼ ▼ ▼ ▼ ▼ ▼ │ │
│ ┌──────────────────────┐ │ │
│ │ Complex Multiply │ │ │
│ │ W × B (signed) │ │ │
│ │ 16-bit intermediate │ │ │
│ └──────────┬───────────┘ │ │
│ │ │ │
│ ▼ ▼ │
│ ┌──────────────────────────────────┐ │
│ │ Butterfly Equations: │ │
│ │ A' = A + (W×B) │ │

│ │ │
│ ┌──────────▼──────────┐ │

│ │ │

### Complex Divider Unit
┌──────────────────────────────────────────────────────────────┐
│ Complex Divider │
├──────────────────────────────────────────────────────────────┤
│ │
│ Y_real ────┐ │
│ Y_imag ────┼────────────────────────────────┐ │
│ │ │ │
│ X_real ────┼────────────────────────┐ │ │
│ X_imag ────┼────────────────────┐ │ │ │
│ │ │ │ │ │
│ ▼ ▼ ▼ ▼ │
│ ┌──────────────────────────────────────────────┐ │
│ │ Numerator Real = Yr×Xr + Yi×Xi │ │
│ │ Numerator Imag = Yi×Xr - Yr×Xi │ │
│ │ Denominator = Xr² + Xi² │ │
│ │ (all 16-bit intermediate) │ │
│ └──────────────────┬───────────────────────────┘ │
│ │ │
│ ▼ │
│ ┌──────────────────────────────────────────────┐ │
│ │ Division Unit (integer) │ │
│ │ Hr = Numerator_Real / Denominator │ │
│ │ Hi = Numerator_Imag / Denominator │ │
│ │ Divide-by-zero → 0xFF, set flag │ │
│ └──────────────────┬───────────────────────────┘ │
│ │ │
│ ▼ │
│ ┌──────────────────────────────────────────────┐ │
│ │ Saturate to 8-bit signed │ │
│ │ (clamp to ±127) │ │
│ └──────────────────┬───────────────────────────┘ │
│ │ │
│ ┌───────────▼───────────┐ │
│ │ H_real (8-bit) │ │
│ │ H_imag (8-bit) │ │
│ │ Divide-by-zero flag │ │
│ └───────────────────────┘ │
└──────────────────────────────────────────────────────────────┘
