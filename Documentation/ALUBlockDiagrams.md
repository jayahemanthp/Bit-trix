## Functional Block Diagrams

### Butterfly Unit (Radix-2 DIT)
┌──────────────────────────────────────────────────────────────┐<br>
│ Butterfly Unit │
├──────────────────────────────────────────────────────────────┤<br>
│ │<br>
│ A_real ────┐ │<br>
│ A_imag ────┼────────────────────────┐ │<br>
│ │ │ │<br>
│ B_real ────┼──┐ │ │<br>
│ B_imag ────┼──┼──┐ │ │<br>
│ │ │ │ │ │<br>
│ W_real ────┼──┼──┼──┐ │ │<br>
│ W_imag ────┼──┼──┼──┼──┐ │ │<br>
│ │ │ │ │ │ │ │<br>
│ inv ───────┼──┼──┼──┼──┼──┐ │ │<br>
│ │ │ │ │ │ │ │ │<br>
│ ▼ ▼ ▼ ▼ ▼ ▼ │ │<br>
│ ┌──────────────────────┐ │ │<br>
│ │ Complex Multiply │ │ │<br>
│ │ W × B (signed) │ │ │<br>
│ │ 16-bit intermediate │ │ │<br>
│ └──────────┬───────────┘ │ │<br>
│ │ │ │<br>
│ ▼ ▼ │<br>
│ ┌──────────────────────────────────┐ │<br>
│ │ Butterfly Equations: │ │<br>
│ │ A' = A + (W×B) │ │<br>
│ │ B' = A - (W×B) │ │<br>
│ └──────────┬───────────────────────┘ │<br>
│ │ │<br>
│ ┌──────────▼──────────┐ │<br>
│ │ Saturate to 8-bit │ │<br>
│ │ (clamp to ±127) │ │<br>
│ └──────────┬──────────┘ │<br>
│ │ │<br>
│ ┌──────────▼──────────┐ │<br>
│ │ Output: A'_real, │ │<br>
│ │ A'_imag, B'_real, │ │<br>
│ │ B'_imag │ │<br>
│ └─────────────────────┘ │<br>
└──────────────────────────────────────────────────────────────┘<br>

### Complex Divider Unit
┌──────────────────────────────────────────────────────────────┐<br>
│ Complex Divider │<br>
├──────────────────────────────────────────────────────────────┤<br>
│ │<br>
│ Y_real ────┐ │<br>
│ Y_imag ────┼────────────────────────────────┐ │<br>
│ │ │ │<br>
│ X_real ────┼────────────────────────┐ │ │<br>
│ X_imag ────┼────────────────────┐ │ │ │<br>
│ │ │ │ │ │<br>
│ ▼ ▼ ▼ ▼ │<br>
│ ┌──────────────────────────────────────────────┐ │<br>
│ │ Numerator Real = Yr×Xr + Yi×Xi │ │<br>
│ │ Numerator Imag = Yi×Xr - Yr×Xi │ │<br>
│ │ Denominator = Xr² + Xi² │ │<br>
│ │ (all 16-bit intermediate) │ │<br>
│ └──────────────────┬───────────────────────────┘ │<br>
│ │ │<br>
│ ▼ │<br>
│ ┌──────────────────────────────────────────────┐ │<br>
│ │ Division Unit (integer) │ │<br>
│ │ Hr = Numerator_Real / Denominator │ │<br>
│ │ Hi = Numerator_Imag / Denominator │ │<br>
│ │ Divide-by-zero → 0xFF, set flag │ │<br>
│ └──────────────────┬───────────────────────────┘ │<br>
│ │ │<br>
│ ▼ │<br>
│ ┌──────────────────────────────────────────────┐ │<br>
│ │ Saturate to 8-bit signed │ │<br>
│ │ (clamp to ±127) │ │<br>
│ └──────────────────┬───────────────────────────┘ │<br>
│ │ │<br>
│ ┌───────────▼───────────┐ │<br>
│ │ H_real (8-bit) │ │<br>
│ │ H_imag (8-bit) │ │<br>
│ │ Divide-by-zero flag │ │<br>
│ └───────────────────────┘ │<br>
└──────────────────────────────────────────────────────────────┘<br>
