```diagram
┌──────────────────────────────────────────────────────────────┐
│ Butterfly Unit                                               │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│ A_real ────┐                                                 │
│ A_imag ────┼────────────────────────┐                        │
│            │                        │                        │
│ B_real ────┼──┐ │ │
│ B_imag ────┼──┼──┐ │ │
│ │ │ │ │ │
│ W_real ────┼──┼──┼──┐ │ │
│ W_imag ────┼──┼──┼──┼──┐ │ │
│ │ │ │ │ │ │ │
│ inv ───────┼──┼──┼──┼──┼──┐ │ │
│ │ │ │ │ │ │ │ │
│ ▼ ▼ ▼ ▼ ▼ ▼ │ │
│ ┌──────────────────────┐ │ │
│ │ Complex Multiply │ │ │
│ │ W × B (signed) │ │ │
│ │ 16-bit intermediate │ │ │
│ └──────────┬───────────┘ │ │
│ │ │ │
│ ▼ ▼ │
│ ┌──────────────────────────────────┐ │
│ │ Butterfly Equations: │ │
│ │ A' = A + (W×B) │ │

│ │ │<br>

│ │ │<br>
│ ┌──────────▼──────────┐ │<br>

│ │ B'_imag │ │<br>

```
### Complex Divider Unit
```diagram
┌──────────────────────────────────────────────────────────────┐<br>
│ Complex Divider                                             │<br>
├──────────────────────────────────────────────────────────────┤<br>
│ │<br>
│ Y_real ────┐ │<br>
│ Y_imag ────┼────────────────────────────────┐ │<br>
│ │ │ │<br>
│ X_real ────┼────────────────────────┐ │ │<br>
│ X_imag ────┼────────────────────┐ │ │ │<br>
│ │ │ │ │ │<br>
│ ▼ ▼ ▼ ▼ │<br>
│ ┌──────────────────────────────────────────────┐ │<br>
│ │ Numerator Real = Yr×Xr + Yi×Xi │ │<br>
│ │ Numerator Imag = Yi×Xr - Yr×Xi │ │<br>
│ │ Denominator = Xr² + Xi² │ │<br>
│ │ (all 16-bit intermediate) │ │<br>
│ └──────────────────┬───────────────────────────┘ │<br>
│ │ │<br>
│ ▼ │<br>
│ ┌──────────────────────────────────────────────┐ │<br>
│ │ Division Unit (integer) │ │<br>
│ │ Hr = Numerator_Real / Denominator │ │<br>
│ │ Hi = Numerator_Imag / Denominator │ │<br>
│ │ Divide-by-zero → 0xFF, set flag │ │<br>
│ └──────────────────┬───────────────────────────┘ │<br>
│ │ │<br>
│ ▼ │<br>
│ ┌──────────────────────────────────────────────┐ │<br>
│ │ Saturate to 8-bit signed │ │<br>
│ │ (clamp to ±127) │ │<br>
│ └──────────────────┬───────────────────────────┘ │<br>
│ │ │<br>
│ ┌───────────▼───────────┐ │<br>
│ │ H_real (8-bit) │ │<br>
│ │ H_imag (8-bit) │ │<br>
│ │ Divide-by-zero flag │ │<br>
│ └───────────────────────┘ │<br>
└──────────────────────────────────────────────────────────────┘<br>
```
